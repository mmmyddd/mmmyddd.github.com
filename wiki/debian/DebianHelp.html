Debian 帮助系统

Debian 参考手册

Debian 参考手册可以用一下命令安装: 

sudo apt-get install debian-reference-en debian-reference-zh-cn

安装完成后，直接运行以下命令 

debian-reference

该命令会运行系统默认的浏览器，并且显示已经安装在本机的Debian参考手册

manpages manpages-dev

manpages-zh manpages-posix manpages-posix-dev

linuxinfo

glibc-doc glibc-doc-reference

Debian Info

Debian系统中如果安装额外的info文件（可能是压缩格式）需要运行 install-info 命令。

获取帮助

info install-info man install-info

安装一个info文件， 最简单的格式：

sudo install-info <info-file>
Notes:
  1. <info-file>如果是gzip压缩文件，可以省略.gz后缀名 例如：emacs22.info.gz 可以直接写为 emacs22.info
  2. <info-file>的名称(不包括.info)需要和<info-file>头部指定的文件名一致

例如：如果头部链接指定文件名为eintr

<quote>

INFO-DIR-SECTION Emacs
START-INFO-DIR-ENTRY

* Emacs Lisp Intro: (eintr).
  			A simple introduction to Emacs Lisp programming.
END-INFO-DIR-ENTRY

</quote>
则该文件在info文件的搜索路径中的名字也应该是eintr.info(.gz)
